INSTRUÇÕES DE USO:

O código está funcional, porém, algumas variáveis não estão preparadas para inputs inesperados, 
por isso recomendo fortemente que na hora do teste sejam seguidas as instruções dos menus e que
os inputs sejam respeitados. Novamente reforço que fora dos menus, a qualquer momento o código 
pode apresentar mensagens de erro e handlers de exceções caso receba um input inesperado.

Estas são as únicas instruções, o método main para execução encontra-se no arquivo App.java.
